# 📝 To-Do App

A modern and interactive To-Do List web application built using HTML, CSS, JavaScript, and jQuery.

## 🚀 Live Demo

🔗 https://ganibasha.github.io/Todo-List/

---

## 📌 Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✔ Mark tasks as completed
- ⌨ Add tasks using Enter key
- 🚫 Prevent empty tasks
- 🎨 Modern responsive UI
- ⚡ Dynamic DOM manipulation using jQuery

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- jQuery
